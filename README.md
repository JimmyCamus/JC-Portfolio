## Install process

### Clone repo

```
$ git clone https://github.com/JimmyCamus/JC-Portfolio.git
```

### Move to project folder

```
$ cd JC-Portfolio
```

### Install node module

```
$ npm install
```

### Env config

You can view the ".env.example" file to see which env variables the project needs.

### Start developer server

```
$ npm run dev
```
